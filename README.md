# Our Drawing Board

## Project Overview
This repository contains the backend server for the "Our Drawing Board" project. The server is responsible for handling all backend logic and API endpoints.

## Important Note
The frontend for this project is no longer maintained in this repository. It is now hosted and actively developed at:

[https://romeomadeit.work/drawing-board](https://romeomadeit.work/drawing-board)

If you are looking for the user interface or client-side code, please visit the above link.

## Repository Contents
- `server.js`: The main backend server file. This is the only useful and maintained part of this repository.
- `public/`: Contains legacy frontend files (no longer in use).
- `package.json`: Project dependencies and scripts.

## Usage
To run the backend server locally:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```
